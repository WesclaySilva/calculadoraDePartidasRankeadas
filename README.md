# Calculadora de Partidas Rankeadas

Desafio feito durante o curso de Lógica de Programação com JavaScript na DIO.

# O que faz

Recebe o número de vitórias e derrotas de um jogador, calcula o saldo e retorna o nível no ranking.

# Níveis

Saldo | Nível |

| Ate 10 | Ferro |
| 11 a 20 | Bronze |
| 21 a 50 | Prata |
| 51 a 80 | Ouro |
| 81 a 90 | Diamante |
| 91 a 100 | Lendario |
| 101 ou mais | Imortal |
